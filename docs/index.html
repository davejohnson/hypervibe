Workflow

A predictable, auditable pattern. Preview first. Confirm when you mean it.

1. Connect providers
Store credentials encrypted locally. Scope tokens by domain/repo when you can. Use connection_create + connection_verify.
2. Plan and bootstrap
Preview with infra_plan, then run stack_bootstrap confirm=true to converge.
3. Protected production
Add policy gates with project_policy_set. For protected envs, use approvals: approval_request_create → approval_request_approve → deploy with approvalId.
4. Move your database
Use db_migrate_provider phases (plan, copy, cutover), then rollback with db_migrate_provider_rollback if needed.
