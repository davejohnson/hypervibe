Get started

Infraprint is an MCP server that gives Claude Code direct access to your infrastructure providers. One command to install, then manage everything through conversation.

Requires Node 20+ and Claude Code.

Terminal
claude mcp add infraprint -- npx infraprint

That's it. Start a Claude Code session and connect your first provider:

Claude Code
# Just ask Claude in natural language:

"Connect my Railway account"
# Claude will walk you through creating a token and connecting it.

"Create a new project called my-app with postgres"
# Claude provisions everything and wires up DATABASE_URL.

"Deploy to staging"
# Your app is live.
